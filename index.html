Introdução ao HTML

O que é HTML?

HTML, ou HyperText Markup Language, é a linguagem padrão para criar páginas web. Foi desenvolvido para definir a estrutura básica do conteúdo de uma página.

Estrutura de um Documento HTML

Um documento HTML é composto por uma série de elementos que definem diferentes partes do conteúdo.

Elementos de Cabeçalho

Os elementos de cabeçalho, h1 a h6, são usados para definir títulos e subtítulos em uma página.

Outros Elementos Importantes

Além dos cabeçalhos, existem vários outros elementos que ajudam a formatar o conteúdo:

  • Parágrafos (

    ): Utilizados para agrupar blocos de texto.

  • Negrito (): Destaca o texto em negrito.
  • Itálico (): Destaca o texto em itálico.
  • Sublinhado (): Destaca o texto sublinhado.
  • Texto marcado (): Destaca o texto como se fosse marcado com um marcador.
  • Texto menor (): Exibe o texto em tamanho reduzido.
  • Texto excluído (): Mostra o texto como se tivesse sido deletado.
  • Subscrito (): Exibe o texto como subscrito.
  • Sobrescrito (): Exibe o texto como sobrescrito.
  • Bloco de citação (
    ): Utilizado para citar trechos de outros documentos.
  • Fonte (): Permite alterar o tipo, tamanho e cor da fonte.
  • Link (): Cria um hiperlink para outra página ou recurso.
  • Linha horizontal (
    ):     Insere uma linha horizontal para separar o conteúdo.
